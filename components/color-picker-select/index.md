## 简介

结合react-color的SketchPicker实现


## API

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| className | 类名 | string | - | |
| defaultValue | 初始默认颜色 | string |  |  |
| value | 颜色值 | string |  |  |
| disabled | 是否禁用 | boolean | false |  |
| allowClear | 清楚当前value值 | boolean | true |  |
| onChange | 变化时回调函数 | Function(color: string) |  |  |

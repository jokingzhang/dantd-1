`dantd` 和 `antd`保持一致，默认文案是英文，如果需要使用其他语言，可以参考下面的方案。

## IntlProvider

```
import {IntlProvider as DantdIntlProvider} from 'dantdv3';

const language = isEn ? 'en-US' : 'zh-CN';

return (
  <DantdIntlProvider locale={language}>
    <App />
  </DantdIntlProvider>
);
```

目前支持以下语言：

| 语言 | key |
| :-- | :-- |
| 英文（默认） | en-US |
| 中文 | zh-CN |
